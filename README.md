# 🛒 Customer Shopping Behavior Analysis  

🚀 **End-to-End Data Analytics Project | Python • SQL • Power BI**

---

## 📌 Problem Statement  
Understanding customer behavior is crucial for increasing revenue and improving customer retention.  
This project analyzes real-world transactional data to extract actionable insights for business growth.

---

## 🎯 Key Objectives  
- Analyze customer purchasing patterns  
- Identify high-value customer segments  
- Evaluate product performance  
- Understand the impact of discounts & subscriptions  

---

## 📊 Project Highlights  
✔️ Analyzed **3,900+ transactions** across multiple product categories  
✔️ Built **data pipeline (Python → PostgreSQL → Power BI)**  
✔️ Performed **business-driven SQL analysis**  
✔️ Designed an **interactive dashboard** for decision-making  

---

## 🧠 Key Insights  
- 💰 Subscription users contribute higher overall revenue  
- 🛍️ Discount usage increases sales but affects profit margins  
- ⭐ Top-rated products drive repeat purchases  
- 🚚 Express shipping users tend to spend more  
- 👥 Loyal customers form the highest revenue segment  

---

| Feature Type        | Examples |

| Purchase Details   | Item, Category, Amount |
| Behavior Metrics   | Frequency, Discounts, Ratings |
